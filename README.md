# AI-Powered Code Review Assistant

An AI-powered web application that helps developers analyze and review code by providing intelligent suggestions, identifying potential issues, and improving code quality.

## Features

* AI-powered code analysis
* Automated code review suggestions
* Modern and responsive user interface
* Fast and intuitive user experience
* Built using modern web technologies

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Supabase

## Installation

### Clone the Repository

```bash
git clone https://github.com/Kamalsharma209/AI-Powered-Code-Review-Assistant.git
cd AI-Powered-Code-Review-Assistant
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:3000
```

## Build for Production

```bash
npm run build
npm start
```
